Submit a plugin

Share your integration with the Qanot AI community. All submissions are security-reviewed before inclusion in the registry.

1 Create your plugin repo

Your repo should follow this structure:

qanot-plugin-your-name/
  plugin.json      # manifest
  plugin.py        # QanotPlugin class
  README.md        # usage docs

Example plugin.json:

{
  "name": "your_plugin",
  "version": "1.0.0",
  "description": "What your plugin does",
  "author": "Your Name",
  "dependencies": ["aiohttp"],
  "required_config": ["api_key"]
}
2 Test locally 
# install from your repo
$ qanot plugin install https://github.com/you/qanot-plugin-name.git

# security scan
$ qanot plugin scan your_name

# run the bot
$ qanot start
3 Submit for review

Option A: Fork QANOT/qanot-plugins, add your entry to index.json, open a PR.

Option B: Use the quick form below — it creates a GitHub Issue for review.

Security Requirements
  • No install.sh, .exe, .bat, or shell scripts
  • No eval(), exec(), os.system()
  • No embedded secrets or .env files
  • Dependencies from approved allowlist only
  • HTTPS git URLs only
