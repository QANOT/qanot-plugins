Submit a Plugin

Share your plugin with the Qanot AI community. All submissions are security-reviewed before inclusion.

1 Create your plugin repo

Your repo should follow this structure:

qanot-plugin-your-name/
  plugin.json      # Required: manifest
  plugin.py        # Required: QanotPlugin class
  README.md        # Recommended: usage docs

Example plugin.json:

{
  "name": "your_plugin",
  "version": "1.0.0",
  "description": "What your plugin does",
  "author": "Your Name",
  "dependencies": ["aiohttp"],
  "required_config": ["api_key"]
}

2 Test locally

Install and scan your plugin before submitting:

# Install from local or git
qanot plugin install https://github.com/you/qanot-plugin-your-name.git

# Security scan
qanot plugin scan your_name

# Verify it works
qanot start

3 Submit for review

Two ways to submit:

  • GitHub PR — Fork QANOT/qanot-plugins, add your entry to index.json, open a PR
  • Quick Submit — Fill the form below and we'll create the PR for you

Security Requirements

  • No install.sh, .exe, .bat, or shell scripts
  • No eval(), exec(), os.system() unless justified
  • No embedded secrets or .env files
  • Dependencies must be from the approved allowlist
  • HTTPS git URLs only
